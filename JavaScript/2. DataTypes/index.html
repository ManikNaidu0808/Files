Type Coercion –Type Coercion is the automatic conversion of “Values” from one data type to another.
–It is a fundamental part of JavaScript and can be used to make code more readable and efficient.
–There are 2 types of coercion in JS: Implicit and Explicit.
–Implicit Coercion happens automatically, while explicit coercion is done manually by the programmer.
–NOTE– It’s worth noting that type coercion can lead to unexpected results, so it’s essential to be aware of how JS handles these situations
