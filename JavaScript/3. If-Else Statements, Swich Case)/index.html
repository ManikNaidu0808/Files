Ternary Operator if-else Statement
Syntax: condition ? valueIfTrue : valueIfFalse Syntax: if (condition) { code to execute if true } else { code to execute if false }
Short and concise way to write a simple if-else statement More versatile, can be used for more complex conditions
Best for simple conditions and when you only need to return one value Can be used for multiple conditions and can execute different code based on the outcome
Limited to only one condition Can handle multiple conditions
Limited to only one value return Can handle multiple return values
